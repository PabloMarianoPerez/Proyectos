
git add . -> agrega todos los archivos para ser comiteados.
git add index.html -> agrega el archivo a ser comiteado.
git commit -m "mensaje" o git commit -am "mensaje" -> comitea los archivos al repositorio.
git init -> inicializo el repositorio local. Se rastrean todos los archivos de esta carpeta.
git ls -> ver contenido de la carpeta correspondiente.
git status -> ver el estado del repositorio.
